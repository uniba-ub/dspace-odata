OData-Query-Generator for Dspace (Alpha)

Dieses Tool parst das OData-Schema und hilf bei der Generierung von Querys.

Eine Einschränkung der angezeigten Entitäten und Attribute der Entitäten ist über eine Whitelist bzw. Blacklist möglich. Über ein Alias können andere (menschenlesbarere) Bezeichnungen von Entitäten und Attribute für die Anzeige Alias vergeben werden. Infos dazu finden sich im Javascript-Code.

  • Wählen Sie hier die gewünschte Optionen aus. Der Link wird anschließend unten generiert.
  • Über den zweiten Button können sie die Existenz des eingegebenen Identifikators prüfen.
  • Außerdem können Sie einen Query in das Formular übertragen
