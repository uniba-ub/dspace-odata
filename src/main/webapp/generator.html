OData-Query-Generator for Dspace

Dieses Tool parst das OData-Schema, stellt eine Übersicht über entsprechende Query-Möglichkeiten zusammen und erstellt einen Query.

Eine Einschränkung der angezeigten Entitäten und Attribute der Entitäten ist über eine Whitelist bzw. Blacklist möglich. Andere (menschenlesbarere) Bezeichnungen von Entitäten und Attribute für die Anzeige können über ein Alias vergeben werden.

Wählen Sie hier die gewünschte Option aus. Der Link wird anschließend unten generiert- Über den zweiten Button können sie die Existenz des eingegebenen Identifikators prüfen.
